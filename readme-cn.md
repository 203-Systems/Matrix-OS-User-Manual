# Matrix OS 用户手册

这是Matrix OS用户手册的源文件。手册里的参考设备是203 Matrix系列控制器。

编译后的PDF文件在/PDF文件夹里

你可以在此仓库的不同分支中查看此文档的旧版本。

## 构建
 目前只在Windows上测试过

 1. 安装 [Pandoc](https://pandoc.org/installing.html)
 2. 安装 [MiKTex](https://miktex.org/download) - 这会安装pdflatex
3. 转到Source\\\<语言>
4. 运行make
